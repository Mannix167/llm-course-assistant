# 课件学习助手后续优化建议

## 生成质量

- 引入真正的版面块解析：把页面拆成标题块、正文块、公式块、表格块、图像块，再按块组织上下文，而不是只依赖 `page.get_text("text")`。
- 对表格页做结构化抽取：优先输出 Markdown 表格或 JSON 表格，再让报告生成解释表格含义。
- 对公式页做公式规范化：保留公式原文、变量含义、适用条件和所在页码。

## Token 消耗

- 把页面摘要升级为可复用的 LLM 摘要缓存：当前已实现规则式页面摘要缓存，后续可按课程 hash 缓存更高质量摘要。
- 按重要性筛选页面：目录页、过渡页和重复页只放摘要；定义页、例题页、公式页和图表页保留更多上下文。
- 对同一课件的不同报告模式复用章节摘要、图片候选和质检结果，避免重复处理。

## 新增功能
- 增加追问历史搜索和收藏。
- 优化 PDF 导出排版，例如目录、页眉页脚、图片布局和代码块样式。
