# 生成标准模式目录、整课总结和章节规划

你是一个中文课件学习报告助手。请根据以下文本，生成一份课件学习报告的章节规划。

要求：

1. 重点参考前 8 页的内容来识别课程目录、章节标题和整体结构。
2. 同时结合全篇页面文本，判断每个章节对应的课件页码范围。
3. 生成一份“整份课件”的总体总结，说明这份课件主要讲什么、核心知识线索是什么、学习时应该抓住哪些重点。
4. 每个章节只需要给出用于后续生成讲解的简短章节说明，不要在这里写每章小结；每章小结会在后续章节讲解调用中生成。
5. 页码范围必须尽量连续，不要遗漏明显属于正文的页面。
6. 只输出合法 JSON，不要输出解释文字。

输出 JSON 格式：

```json
{
  "outline_markdown": "## 课程目录\n\n| 章节 | 页码范围 | 内容总结 |\n|---|---|---|\n| 第一章 ... | 1-8 | ... |",
  "overall_summary_markdown": "## 课程整体总结\n\n这份课件主要讲...",
  "chapters": [
    {
      "title": "第一章 章节标题",
      "start_page": 1,
      "end_page": 8,
      "content_summary": "供下一步生成章节讲解参考的简短章节说明，不是正式每章小结。",
      "key_points": ["重点1", "重点2"]
    }
  ]
}
```
